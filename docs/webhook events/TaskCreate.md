# TaskCreate

Called whenever a task is created for a contact.

#### Example

```json
{
  "type": "TaskCreate",
  "locationId": "ve9EPM428h8vShlRW1KT",
  "contactId": "nmFmQEsNgz6AVpgLVUJ0",
  "id": "TaskId123",
  "title": "Follow up call",
  "dueDate": "2024-02-01T09:00:00.000Z",
  "completed": false
}
```
