# OpportunityUpdate

Called whenever an opportunity is updated.

#### Example

```json
{
  "type": "OpportunityUpdate",
  "locationId": "ve9EPM428h8vShlRW1KT",
  "id": "9NkT25Vor1v4aQatFsv2",
  "contactId": "nmFmQEsNgz6AVpgLVUJ0",
  "name": "Updated Deal",
  "monetaryValue": 2000,
  "status": "open"
}
```
