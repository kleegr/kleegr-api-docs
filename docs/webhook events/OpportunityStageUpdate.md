# OpportunityStageUpdate

Called whenever an opportunity's pipeline stage is updated.

#### Example

```json
{
  "type": "OpportunityStageUpdate",
  "locationId": "ve9EPM428h8vShlRW1KT",
  "id": "9NkT25Vor1v4aQatFsv2",
  "pipelineId": "WH9lbGUoY7ZNoHi9q1SK",
  "pipelineStageId": "NewStageId"
}
```
