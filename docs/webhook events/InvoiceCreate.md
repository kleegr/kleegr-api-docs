# InvoiceCreate

Called whenever an invoice is created.

#### Example

```json
{
  "type": "InvoiceCreate",
  "locationId": "ve9EPM428h8vShlRW1KT",
  "id": "BqTwX8QFwXzpegMve9EQ",
  "contactId": "nmFmQEsNgz6AVpgLVUJ0",
  "status": "draft",
  "amountDue": 150.00,
  "currency": "USD"
}
```
