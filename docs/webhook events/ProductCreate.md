# ProductCreate

Called whenever a product is created.

#### Example

```json
{
  "type": "ProductCreate",
  "locationId": "ve9EPM428h8vShlRW1KT",
  "id": "prod_abc123",
  "name": "Premium Plan",
  "description": "Full access plan",
  "statementDescriptor": "KLEEGR-PREMIUM"
}
```
