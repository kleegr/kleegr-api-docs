# InvoiceSent

Called whenever an invoice is sent to a contact.

#### Example

```json
{
  "type": "InvoiceSent",
  "locationId": "ve9EPM428h8vShlRW1KT",
  "id": "BqTwX8QFwXzpegMve9EQ",
  "contactId": "nmFmQEsNgz6AVpgLVUJ0",
  "status": "sent",
  "amountDue": 150.00
}
```
