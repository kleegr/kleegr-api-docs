# RecordUpdate

Triggered when a custom object record or business is updated.

#### Example

```json
{
  "id": "679b8f9bde6a0c356a0311b3",
  "locationId": "eHy2cOSZxMQzQ6Yyvl8P",
  "timestamp": "2025-02-10T08:26:05.961Z",
  "owners": ["60d5ec49f72b2a001f5f9d91"],
  "properties": [{"key": "pet_name", "valueString": "buddy-updated"}]
}
```
