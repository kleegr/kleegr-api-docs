# UserCreate

Called whenever a new user is created in a Kleegr CRM location.

#### Example

```json
{
  "type": "UserCreate",
  "locationId": "ve9EPM428h8vShlRW1KT",
  "id": "YlWd2wuCAZQzh2cH1fVZ",
  "firstName": "Jane",
  "lastName": "Smith",
  "email": "jane@example.com",
  "role": "user",
  "type": "account"
}
```
