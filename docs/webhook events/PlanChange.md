# PlanChange

Called whenever an app's plan/subscription changes for a location.

#### Example

```json
{
  "type": "PlanChange",
  "locationId": "ve9EPM428h8vShlRW1KT",
  "appId": "ve9EPM428h8vShlRW1KT",
  "planId": "plan_pro",
  "previousPlanId": "plan_starter"
}
```
