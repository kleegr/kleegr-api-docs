# OrderCreate

Called whenever a new order is placed.

#### Example

```json
{
  "type": "OrderCreate",
  "locationId": "ve9EPM428h8vShlRW1KT",
  "id": "ORD_abc123",
  "contactId": "nmFmQEsNgz6AVpgLVUJ0",
  "status": "pending",
  "totalAmount": 99.99,
  "currency": "USD"
}
```
