Quick start
1
Register an OAuth app at marketplace.gohighlevel.com
2
Redirect the user to the Authorization URL with your client_id and scopes
3
Exchange the authorization code for an access token
4
Call any API endpoint using the access token
Example request
GET /contacts/search Authorization: Bearer {access_token} Version: 2021-07-28 Content-Type: application/json // 200 OK { "contacts": [...], "total": 142 }
