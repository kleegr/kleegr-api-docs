Full interactive API reference for all Kleegr CRM endpoints. Each page loads the full OpenAPI spec with request/response schemas, code samples, and a try-it-out panel powered by Scalar.

All endpoints are served at `https://services.leadconnectorhq.com`.

Required headers on every request:
- `Authorization: Bearer ACCESS_TOKEN`
- `Version: 2021-07-28`
