Kleegr developer integrations are powered by HighLevel APIs. Certain technical references are retained for integration accuracy.

## What is retained and why

| Reference | Reason |
|-----------|--------|
| OAuth registration at `marketplace.gohighlevel.com` | Required to get OAuth credentials |
| White-label OAuth URL `marketplace.leadconnectorhq.com` | Required for white-label flows |
| API base URL `services.leadconnectorhq.com` | All API calls go here |
| OAuth scope strings (e.g. `contacts.readonly`) | Part of the API contract |
| All endpoint paths (e.g. `/contacts/search`) | The actual API URLs |
| Field names and schema definitions | Required for correct integrations |
| Webhook signature public key | Required for verification |
| Rate limit headers (`X-RateLimit-*`) | Documented for monitoring |
