---
title: InboundMessage
description: Webhook when a contact sends a message to a Kleegr CRM location.
---

Channels: SMS, Email, Call, Voicemail, GMB, Facebook, Instagram, Live Chat.

## SMS
```json
{"type":"InboundMessage","locationId":"l1C08ntBrFjLS0elLIYU","body":"Hi, I have a question","contactId":"cI08i1Bls3iTB9bKgFJh","conversationId":"fcanlLgpbQgQhderivVs","direction":"inbound","messageType":"SMS","status":"delivered","dateAdded":"2024-01-15T10:00:00.000Z"}
```

## Call
```json
{"type":"InboundMessage","locationId":"l1C08ntBrFjLS0elLIYU","contactId":"cI08i1Bls3iTB9bKgFJh","direction":"inbound","messageType":"CALL","status":"completed","callDuration":180,"callStatus":"completed"}
```

**Required scope:** `conversations/message.readonly`
