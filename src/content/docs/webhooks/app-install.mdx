## Location install

```json
{
  "type": "INSTALL",
  "appId": "ve9EPM428h8vShlRW1KT",
  "locationId": "otg8dTQqGLh3Q6iQI55w",
  "companyId": "otg8dTQqGLh3Q6iQI55w",
  "planId": "plan_starter",
  "trial": {"onTrial": false}
}
```

## AppUninstall

Same structure, `type: "UNINSTALL"`.
