## ContactCreate

```json
{"type":"ContactCreate","locationId":"abc","id":"xyz","firstName":"Jane","lastName":"Smith","email":"jane@example.com","phone":"+15551234567","tags":["lead"],"dnd":false,"dateAdded":"2024-01-15T10:00:00.000Z"}
```

## ContactUpdate

Same schema, `type: "ContactUpdate"`, only changed fields present.

## ContactDelete

```json
{"type":"ContactDelete","locationId":"abc","id":"xyz"}
```

## ContactTagUpdate

```json
{"type":"ContactTagUpdate","locationId":"abc","id":"xyz","tags":["lead","hot"]}
```
