---
title: Contact Events
description: ContactCreate, ContactUpdate, ContactDelete, ContactTagUpdate payloads.
---

## ContactCreate
```json
{"type":"ContactCreate","locationId":"abc","id":"xyz","firstName":"Jane","lastName":"Smith","email":"jane@example.com","tags":["lead"],"dnd":false}
```

## ContactUpdate
Same schema, `type:"ContactUpdate"`, only changed fields present.

## ContactDelete
```json
{"type":"ContactDelete","locationId":"abc","id":"xyz"}
```

## ContactTagUpdate
```json
{"type":"ContactTagUpdate","locationId":"abc","id":"xyz","tags":["lead","hot"]}
```

**Required scope:** `contacts.readonly`
