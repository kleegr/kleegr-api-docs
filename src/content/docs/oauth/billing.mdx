---
title: External Billing
description: Confirm external payments for paid marketplace apps.
---

## Prerequisites

At [marketplace.gohighlevel.com](https://marketplace.gohighlevel.com): Business model = Paid, External Billing enabled, Billing URL configured.

## Confirm payment endpoint

```bash
curl -X POST https://services.leadconnectorhq.com/oauth/billing/webhook \
  -H "x-ghl-client-key: YOUR_KEY" \
  -H "x-ghl-client-secret: YOUR_SECRET" \
  -H "Content-Type: application/json" \
  -d '{"clientId":"ID","authType":"location","locationId":"LOC","amount":49,"status":"COMPLETED","paymentType":"recurring"}'
```

| Field | Values |
|-------|--------|
| `authType` | `location` or `company` |
| `status` | `COMPLETED` or `FAILED` |
| `paymentType` | `one_time` or `recurring` |
