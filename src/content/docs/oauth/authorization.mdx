import { Steps, Aside } from '@astrojs/starlight/components';

<Steps>

1. **Register an OAuth app** at [marketplace.gohighlevel.com](https://marketplace.gohighlevel.com) under My Apps.

2. **Build the Authorization URL**

   ```
   https://marketplace.gohighlevel.com/oauth/chooselocation?
   response_type=code&redirect_uri=https://yourapp.com/callback&
   client_id=YOUR_CLIENT_ID&scope=contacts.readonly
   ```

   White-label: use `marketplace.leadconnectorhq.com` instead.

3. **Handle the callback** — user is sent to `https://yourapp.com/callback?code=AUTH_CODE`

4. **Exchange code for tokens**

   ```bash
   curl -X POST https://services.leadconnectorhq.com/oauth/token \
     -H "Content-Type: application/x-www-form-urlencoded" \
     -d "client_id=YOUR_CLIENT_ID&client_secret=YOUR_SECRET" \
     -d "grant_type=authorization_code&code=AUTH_CODE" \
     -d "redirect_uri=https://yourapp.com/callback"
   ```

5. **Refresh the token** (tokens expire after 24 hours)

   ```bash
   curl -X POST https://services.leadconnectorhq.com/oauth/token \
     -H "Content-Type: application/x-www-form-urlencoded" \
     -d "client_id=YOUR_CLIENT_ID&client_secret=YOUR_SECRET" \
     -d "grant_type=refresh_token&refresh_token=YOUR_REFRESH_TOKEN"
   ```

</Steps>

## Rate limits

| Limit | Value |
|-------|-------|
| Burst | 100 req / 10 sec per app per location |
| Daily | 200,000 req / day per app per location |
