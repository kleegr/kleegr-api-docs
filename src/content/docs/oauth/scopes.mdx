<Aside type="caution">Scope strings are part of the API contract. Use them exactly as listed.</Aside>

## Contacts

| Scope | Access |
|-------|--------|
| `contacts.readonly` | Read contact records |
| `contacts.write` | Create and update contacts |

## Conversations

| Scope | Access |
|-------|--------|
| `conversations.readonly` | Read conversations |
| `conversations.write` | Create and update conversations |
| `conversations/message.readonly` | Read messages, recordings, transcriptions |
| `conversations/message.write` | Send messages, add call logs |

## Calendars

| Scope | Access |

| `calendars.readonly` | Read calendars |
| `calendars.write` | Create and manage calendars |
| `calendars/events.readonly` | Read appointments |
| `calendars/events.write` | Create and update appointments |
