---
title: OAuth Scopes
description: Available OAuth permission scopes for Kleegr API integrations.
---

## Contacts
| `contacts.readonly` | Read contacts | `contacts.write` | Create/update contacts |

## Conversations
| Scope | Access |
|-------|--------|
| `conversations.readonly` | Read conversations |
| `conversations.write` | Create/update conversations |
| `conversations/message.readonly` | Read messages |
| `conversations/message.write` | Send messages |

## Calendars
| Scope | Access |
|-------|--------|
| `calendars.readonly` | Read calendars |
| `calendars.write` | Manage calendars |
| `calendars/events.readonly` | Read appointments |
| `calendars/events.write` | Create/update appointments |

## Opportunities
| `opportunities.readonly` | Read |
| `opportunities.write` | Create/update |

## Invoices & Payments
| `invoices.readonly` | Read invoices |
| `invoices.write` | Manage invoices |
| `payments.readonly` | Read payments |
| `payments.write` | Manage payments |

## Users & Locations
| `users.readonly` | Read users |
| `users.write` | Manage users |
| `locations.readonly` | Read sub-accounts |
| `locations.write` | Update sub-accounts |

## Forms, Surveys, Content
| `forms.readonly` | Forms/submissions |
| `surveys.readonly` | Survey responses |
| `blogs.readonly` / `blogs.write` | Blog management |

## Workflows & Campaigns
| `workflows.readonly` | Read workflows |
| `campaigns.readonly` | Read campaigns |

## Media & Products
| `medias.readonly` / `medias.write` | Media library |
| `products.readonly` / `products.write` | Products |
