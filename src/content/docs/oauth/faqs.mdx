## How long are access tokens valid?

Access tokens are valid for **24 hours**. Use the refresh token to get a new one.

## How long are refresh tokens valid?

Refresh tokens are valid for **1 year** and reset with each use.

## How should I handle token expiry?

1. Make a request with the current `access_token`
2. If you get `401 Unauthorized`, call the token refresh endpoint
3. Store the new tokens and retry the request

Write a wrapper function to handle this for all your API calls.

## What are the rate limits?

- **Burst:** 100 requests per 10 seconds per app per location
- **Daily:** 200,000 requests per day per app per location

Each installed location has its own independent budget.

## How do I listen to webhook events?

1. In your OAuth app settings, set a **Webhook URL**
2. Add required **scopes** for the events you want
3. Have a location admin install and authorize your app
4. You will begin receiving webhook payloads
