## How long are access tokens valid?

**24 hours**. Refresh using the refresh token.

## How long are refresh tokens valid?

**1 year**, reset on each use.

## How do I handle token expiry?

1. Make a request with the current `access_token`
2. On `401 Unauthorized`, call the token refresh endpoint
3. Store new tokens and retry

## Rate limits?

- Burst: 100 req / 10 sec per app per location
- Daily: 200,000 req / day per app per location
